Automata 1

Este autómata recibe una cadena con el siguiente patron y las siguientes condiciones:

  • Una cadena debe comenzar con una letra a o b.
  • Si la cadena comienza con una letra b se ciclará hasta encontrar una letra a o hasta acabar la cadena.
  • Si aparece una letra a despues de una serie de letras b, saltará al siguiente estado (Q1).
  • En el estado Q1, si la cadena continua con la letra a se ciclará hasta encontrar una letra b, acabar la cadena o encontrar una letra que no coincida con el patrón.
  • En el estado Q1, si la siguiente letra es una b, continua hacia el siguiente estado.
  • Si la siguiente letra en el estado Q2 es una a regresará al estado Q1
  • Si la siguiente letra en el estado Q2 es una b avanzará hacia el estado Q3
  • Si la siguiente letra en el estado Q3 es una a o una b, se ciclará hasta terminar la cadena o hasta encontrar una letra que no corresponda al patrón.
  • Si cualquiera de estas condiciones no se cumple, el patrón será rechazado.
